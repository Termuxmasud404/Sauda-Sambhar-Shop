🛒 Sauda Sambhar Shop

A simple E-commerce website built with HTML, CSS, JavaScript, and Firebase.

✨ Features

- 🏠 Home Page
- 🔐 User Login
- 👨‍💼 Admin Panel
- ➕ Add Product
- 🗑️ Delete Product
- 📦 View Products
- 🔥 Firebase Authentication
- ☁️ Firebase Firestore Database
- 📱 Responsive Design
- 🚀 Vercel Ready

🔥 Firebase Setup

1. Create a Firebase Project.
2. Enable Authentication (Email/Password).
3. Create a Firestore Database.
4. Replace the Firebase configuration in "js/firebase.js".
5. Deploy using Firebase Hosting or Vercel.

🚀 Run Project

Simply open "index.html" using a local web server (such as VS Code Live Server) or deploy the project to Vercel/Firebase Hosting.
